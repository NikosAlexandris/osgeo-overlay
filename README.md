# OSGeo Overlay [Experimental]

A personal Gentoo/Funtoo Linux overlay aiming to host up-to-date
[OSGeo](http://www.osgeo.org/) software. Namely,

* [GDAL](http://www.gdal.org/)
* [GRASS GIS](https://grass.osgeo.org/)
* ...
